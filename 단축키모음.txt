윈도우 명령어 및 단축키 

==> 단축키 <==

win + r		실행 (run)
win + e		탐색기 (explore)
win + d		바탕화면 보기 (desktop)

win + ←		현재 창 왼쪽으로 붙이기
win + →		현재 창 오른쪽으로 붙이기
win + ↑		현재 창 최대화 or 왼쪽/오른쪽에서 위로 배치
win + ↓		현재 창 원래대로/최소화 or 왼쪽/오른쪽에서 아래로 배치
win + .		이모지/특수문자 입력 모음

alt + d		탐색기 주소창으로 커서 이동



==> 명령어 <==

cmd		명령 프롬프트 창 호출
notepad		메모장
calc		계산기

sysdm.cpl	시스템 속성 제어판 (환경 변수 및 컴퓨터 이름)
appwiz.cpl	프로그램 추가/제거



==> 웹 브라우저 단축키 <==
ctrl + j	다운로드 목록 보기

alt + d		브라우저 주소창으로 커서 이동
ctrl + shift + delete
		개인정보 삭제 (쿠키 및 캐시 삭제)



==> 이클립스 단축키 <==
ctrl + space	자동완성
ctrl + shift + c
		현재 줄 혹은 범위 주석처리
ctrl + m	최대화/복구
ctrl + s	저장 (및 컴파일)
ctrl + F11	실행 (메인함수가 없으면 마지막으로 실행한 프로그램 실행)
F11		디버깅, 중단점 설정 후 코드를 한줄씩 실행하기
ctrl + n	새로 만들기 (프로젝트, 패키지, 파일 등)
